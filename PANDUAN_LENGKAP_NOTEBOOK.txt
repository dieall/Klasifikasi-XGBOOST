### LANGKAH 5: Set Threshold (Section 5.2) ⭐ KRUSIAL!

**Berdasarkan analisis di Step 4, pilih salah satu:**

#### OPSI A: Gunakan Suggested Threshold (P5-P95)
Salin nilai P5 dan P95 dari analisis:
```python
thresholds = {
    'Transmission': (85.23, 98.46),  # Dari P5-P95
    'Tinx': (17.8, 19.2),            # Dari P5-P95
    # ... dst
}
```
→ **Hasil:** ~10% data akan "Tidak Layak"

#### OPSI B: Gunakan Standar Perusahaan (RECOMMENDED)
Tanyakan ke QC/Lab untuk spesifikasi riil:
```python
thresholds = {
    'Transmission': (85, 100),
    'Tinx': (17.5, 19.5),
    'RI': (1.485, 1.495),
    # ... sesuai standar perusahaan
}
```
→ **Hasil:** Sesuai dengan standar produksi riil

#### OPSI C: Custom (Untuk Balanced Data)
Jika Anda ingin data seimbang 50:50:
```python
thresholds = {
    'Transmission': (Q25_value, Q75_value),  # Dari Q25-Q75
    # ... dst
}
```
→ **Hasil:** ~50% "Layak", ~50% "Tidak Layak"

## 🚨 TROUBLESHOOTING

### Problem 1: ValueError pada Pie Chart
**Gejala:**
```
ValueError: 'labels' must be of length 'x', not 2
```

**Penyebab:** Data hanya memiliki 1 kelas

**Solusi:**
1. Kembali ke Section 5.1
2. Lihat hasil analisis distribusi
3. Sesuaikan threshold di Section 5.2:
   - Jika semua "Layak" → PERKETAT threshold
   - Jika semua "Tidak Layak" → LONGGARKAN threshold
4. Jalankan ulang dari Section 5.2

### Problem 2: Data Sangat Imbalanced (99:1)
**Gejala:**
```
1    1492
0       7
```

**Penyebab:** Threshold terlalu longgar atau produksi sangat konsisten

**Solusi:**
1. Gunakan threshold yang lebih ketat (P10-P90 atau Q25-Q75)
2. Atau: Terima kenyataan bahwa produksi memang konsisten
3. Lanjutkan dengan teknik SMOTE (sudah disediakan di code sebelumnya)

### Problem 3: Data Sangat Seimbang tapi Tidak Realistis
**Gejala:**
```
1    750
0    749
```
Padahal seharusnya produksi bagus

**Penyebab:** Threshold terlalu ketat

**Solusi:**
Gunakan threshold yang lebih realistis sesuai standar perusahaan

## 📊 CONTOH THRESHOLD YANG BAIK

### Contoh 1: Produksi Bagus (Imbalance 90:10)
```python
thresholds = {
    'Transmission': (85, 100),    # P5-P95
    'Tinx': (17.5, 19.5),
    'RI': (1.485, 1.495),
    'SG': (1.15, 1.20),
    'Acid': (0, 0.5),
    'Sulfur': (0, 0.05),
    'Water': (0, 0.15),
    'Mono': (0, 5),
    'Yellow': (0, 10),
    'EH': (0, 50),
    'Visco': (30, 60),
    'PT': (0, 10)
}
```
→ Hasil: 90% Layak, 10% Tidak Layak (Realistis untuk produksi bagus)

### Contoh 2: Balanced untuk Penelitian (50:50)
```python
# Gunakan Median sebagai threshold
thresholds = {
    'Transmission': (90, 100),    # Q50-Max (50% di atas median)
    'Tinx': (18.5, 19.0),         # Range sempit
    # ... set agak ketat untuk dapat 50:50
}
```
